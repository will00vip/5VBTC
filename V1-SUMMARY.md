## 二、核心算法详解

### 2.1 综合评分系统（0-100分）

#### 评分规则
```
总得分 = 基础形态(30分) + 共振强度(20分) + 趋势强度(15分) 
       + 波动率评估(10分) + 成交量分析(10分) + 技术指标(15分)
```

#### 信号阈值
| 评分区间 | 信号类型 | 操作建议 |
|----------|----------|----------|
| **≥+60分** | 🟢 做多信号 | 可轻仓试探 |
| **-60~+60分** | 🟡 观望 | 等待机会 |
| **≤-60分** | 🔴 做空信号 | 可轻仓试探 |

### 2.2 技术指标库

| 指标 | 周期 | 用途 |
|------|------|------|
| **EMA** | 20/60 | 趋势判断 |
| **RSI** | 14 | 超买超卖检测 |
| **MACD** | 12/26/9 | 动量分析 |
| **KDJ** | 9 | 随机指标 |
| **BOLL** | 20/2σ | 布林带通道 |
| **ATR** | 14 | 波动率测量 |
| **ADX** | 14 | 趋势强度 |

### 2.3 多周期共振策略

```
日线(D1) → 1小时(1H) → 5分钟(5M)
    ↓           ↓           ↓
趋势确认    方向选择     入场点
```

**共振检测逻辑：**
1. **趋势共振**：日线、1H、5M 同方向
2. **RSI共振**：短周期超买/超卖 + 长周期配合
3. **MACD共振**：DIF/DEA 方向一致
4. **放量确认**：当前成交量 > 均量1.2倍

### 2.4 市场状态识别

| 状态 | 判断条件 | 交易策略 |
|------|----------|----------|
| **趋势市** | ADX>25 + 价格偏离MA20>1% | 顺势而为 |
| **震荡市** | 布林带收窄 + ADX<20 | 高抛低吸 |
| **高波动市** | ATR比值>1.5 或 布林宽度>4% | 减少操作 |
| **反转市** | 均线黏合 + 放量 + ADX<20 | 关注反转 |

### 2.5 智能止盈止损

**止损计算：**
```javascript
// 做多止损
stopLoss = min(ATR止损, 支撑止损)

// 做空止损
stopLoss = max(ATR止损, 压力止损)
```

**止盈目标：**
| 目标 | 距离 |
|------|------|
| TP1 | 1倍ATR |
| TP2 | 2倍ATR |
| TP3 | 3倍ATR |
