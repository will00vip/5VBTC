# BTC三步法雷达 - Android App

基于"三步法"交易策略的BTC实时监控与信号推送App。

## 主要功能

### 📊 实时行情监控
- 实时BTC/USDT价格显示
- 24小时涨跌幅和成交量
- 多周期K线图表（1H/4H/日线）
- 支持火币、Binance双API自动切换

### 📈 技术指标矩阵
| 指标 | 功能 |
|------|------|
| RSI | 相对强弱指数，判断超买超卖 |
| MACD | 趋势判断，金叉死叉信号 |
| KDJ | 随机指标，敏感买卖点 |
| BOLL | 布林带，支撑压力位 |
| 趋势线 | MA5/MA10/MA30均线 |

### 🎯 三步法信号系统
综合评分0-100分，判定交易机会：
- **≥60分**：做多信号（绿色）
- **≤40分**：做空信号（红色）
- **40-60分**：等待观察

### 🔔 24小时后台监控
- **3分钟自动检测**：持续监控市场机会
- **推送通知**：发现信号时即时推送
- **开机自启**：设备重启后自动恢复监控
- **低电量优化**：智能省电，不影响日常使用

### 📜 历史信号记录
- 自动保存所有信号历史
- 显示入场点位和信号时间
- 可清空重置

## 安装使用

### APK下载
直接安装 `btc-radar-v1.0.0.apk`

### 权限说明
- **网络权限**：获取实时行情数据
- **后台运行**：24小时监控信号
- **自启动**：开机自动开启监控
- **通知权限**：推送交易信号提醒

## 技术架构

```
App (HTML/CSS/JS)
    ↓ Capacitor
Android (Java)
    ↓
SignalMonitorService (后台Service)
    ↓
Binance/Huobi API
```

### API数据源
1. 火币API（优先）
2. Binance API（备用）
3. 模拟数据（离线兜底）
