# 📂 分支结构说明

## 分支列表

### 1. main (主分支)
- 项目基础框架
- 核心功能实现
- 统一入口和基础 UI

### 2. 3step-trading (三步交易法主分支) ⭐推荐
- **核心策略**：下影插针 → 低点确认 → MACD 配合
- **适用人群**：学习三步交易法的用户
- **功能**：K线图、信号检测、AI评分、VIP系统
- **安全性**：仅使用行情 API，无交易权限

### 3. auto-trade (自动化交易分支)
- **核心策略**：基于三步交易法 + API 自动下单
- **适用人群**：有编程基础的专业交易者
- **功能**：包含火币/Binance API 交易权限
- **风险等级**：⚠️ 高风险 - 包含真实交易功能

### 4. public-share (大众分享版)
- **核心策略**：与 3step-trading 相同
- **适用人群**：普通用户，App 分发
- **功能**：纯净版，无交易权限
- **风险等级**：✅ 低风险 - 仅行情展示

## 如何选择分支

| 使用场景 | 推荐分支 |
|---------|---------|
| 学习交易策略 | `3step-trading` |
| 自动化交易 | `auto-trade` |
| 分发 App 给用户 | `public-share` |
| 参与开发 | `main` |

## 分支维护

- `main`：长期维护，同步最新功能到各分支
- `3step-trading`：三步交易法核心策略开发
- `auto-trade`：自动化交易功能开发
- `public-share`：用户体验优化，定期合并 main
