# BTC三步法专业交易系统 v6.0 - 专业分析版

## 🎯 核心功能

### 📊 120根K线图表
- **显示范围**：最近120根K线，完整的历史走势
- **颜色方案**：红涨绿跌（符合中国股市习惯）
- **自动缩放**：根据最高价/最低价自动调整价格范围
- **实时价格线**：黄色虚线显示当前收盘价
- **网格线**：灰色水平线辅助价格读取
- **支持周期**：15分钟、1小时、4小时、1天多周期切换

### 📈 MACD副图
- **组成部分**：DIF（蓝色线）、DEA（橙色线）、柱状图（绿/红）
- **参数**：(12,26,9) 标准参数
- **功能**：
  - 检测金叉/死叉（多空转折信号）
  - 识别零轴上下的多空动能
  - 柱状图显示动能强弱

### 📐 BOLL副图（布林带）
- **组成部分**：上轨（红色）、中轨（黄色）、下轨（绿色）
- **参数**：(20,2) 标准参数
- **功能**：
  - 上轨：历史高位，阻力位
  - 中轨：均价线，支撑/阻力
  - 下轨：历史低位，支撑位
  - 填充区域：布林带预期走势区间
  - 价格点：白色点标记当前价格位置

## 🔍 动态信号分析（100分制）

### 6大分析维度

#### 1️⃣ 趋势方向 (20分)
| 状态 | 分数 | 说明 |
|------|------|------|
| 强势多头 | 20分 | ✅ 满足：价格>MA20>MA60，多头强劲 |
| 强势空头 | 20分 | ✅ 满足：价格<MA20<MA60，空头强劲 |
| 多头 | 15分 | ✅ 满足：价格>MA20，趋势向好 |
| 空头 | 15分 | ✅ 满足：价格<MA20，趋势向下 |
| 震荡 | 5分 | ❌ 不满足：价格在MA20附近，无方向 |

**判断标准**：
- **满足条件**（绿色）：价格位于高级别均线上方，适合做多；或在均线下方，适合做空
- **不满足条件**（红色）：价格在均线附近震荡，多空力量均衡，建议观望

#### 2️⃣ MACD动量 (15分)
| 状态 | 分数 | 说明 |
|------|------|------|
| 金叉多头 | 15分 | ✅ 满足：DIF>DEA且柱图>0，多头信号 |
| 死叉空头 | 15分 | ✅ 满足：DIF<DEA且柱图<0，空头信号 |
| 零轴多头 | 10分 | 📊 中性：柱图>0但未金叉，多头但需确认 |
| 零轴空头 | 10分 | 📊 中性：柱图<0但未死叉，空头但需确认 |
| 中线附近 | 5分 | ❌ 不满足：MACD在零轴附近震荡 |

**判断标准**：
- **满足条件**（绿色）：MACD形成金叉或死叉，动能方向明确
- **不满足条件**（红色）：MACD在零轴附近，多空动能均衡
- **中性状态**（灰色）：虽有方向但未确认，需要等待

#### 3️⃣ RSI超买超卖 (15分)
| 状态 | 分数 | 说明 |
|------|------|------|
| 极度超卖 | 15分 | ✅ 满足：RSI<30，反弹机会 |
| 极度超买 | 15分 | ✅ 满足：RSI>70，回调机会 |
| 接近超卖 | 10分 | 📊 中性：RSI 30-40，接近但未极端 |
| 接近超买 | 10分 | 📊 中性：RSI 60-70，接近但未极端 |
| 正常波动 | 5分 | ❌ 不满足：RSI 40-60，正常范围 |

**判断标准**：
- **满足条件**（绿色）：RSI处于极端区间（<30或>70），市场情绪已过度
- **不满足条件**（红色）：RSI在正常范围（40-60），没有明显信号
- **中性状态**（灰色）：接近但未达到极端值

#### 4️⃣ BOLL支撑阻力 (15分)
| 状态 | 分数 | 说明 |
|------|------|------|
| 接近下轨 | 15分 | ✅ 满足：价格<10%布林位置，强支撑 |
| 接近上轨 | 15分 | ✅ 满足：价格>90%布林位置，强阻力 |
| 在下轨附近 | 10分 | 📊 中性：价格10-20%布林位置 |
| 在上轨附近 | 10分 | 📊 中性：价格80-90%布林位置 |
| 中轨附近 | 5分 | ❌ 不满足：价格40-60%布林位置 |

**判断标准**：
- **满足条件**（绿色）：价格接近布林带极值，强支撑/阻力位
- **不满足条件**（红色）：价格在中轨附近，支撑阻力不明显
- **中性状态**（灰色）：虽有支撑阻力但强度一般

#### 5️⃣ K线形态 (20分)
| 状态 | 分数 | 说明 |
|------|------|------|
| 完全符合 | 20分 | ✅ 满足：所有形态条件都达标 |
| 部分符合 | 15分 | 📊 中性：70%以上条件满足 |
| 不符合 | 5分 | ❌ 不满足：大部分条件未达标 |

**具体条件**（做多示例）：
1. **下影插针**：下影线≥实体的2倍，长下影表示有人在低位抄底
2. **低点抬高**：最新低点>前一高点，反转信号
3. **MACD配合**：MACD多头或止跌，动能确认

**判断标准**：
- **满足条件**（绿色）：K线形态完整，反转信号强烈
- **不满足条件**（红色）：缺少关键形态条件，反转不确定

#### 6️⃣ 成交量确认 (15分)
| 状态 | 分数 | 说明 |
|------|------|------|
| 显著放量 | 15分 | ✅ 满足：成交量≥均值的2倍 |
| 温和放量 | 12分 | ✅ 满足：成交量≥均值的1.5倍 |
| 轻微放量 | 8分 | 📊 中性：成交量≥均值的1.2倍 |
| 缩量 | 3分 | ❌ 不满足：成交量<均值 |

**判断标准**：
- **满足条件**（绿色）：成交量明显放大，市场参与度高，信号可靠
- **不满足条件**（红色）：成交量萎缩，市场不活跃，信号不可靠
- **中性状态**（灰色）：成交量一般，信号可靠性中等

## 📈 技术指标矩阵

每个指标包含四个信息：

### 1. 指标值
- 显示具体数值（如 RSI: 45）

### 2. 指标名称
- MACD / RSI / KDJ / WR / BOLL / ATR

### 3. 状态指示（右侧彩条）
| 颜色 | 含义 | 说明 |
|-----|------|------|
| 🟢 绿色 | 满足条件 | ✅ 符合交易要求，可作为入场依据 |
| 🔴 红色 | 不满足条件 | ❌ 未达标，建议等待或谨慎操作 |
| ⚪ 灰色 | 中性 | 📊 处于中间状态，无明显方向 |

### 4. 详细备注（下方说明）

#### MACD 备注示例
- ✅ "满足：金叉且多头，看涨" - DIF>DEA且>0
- ❌ "不满足：多头但死叉，警惕" - DIF<DEA但柱图>0
- 📊 "中性：MACD在零轴附近震荡" - 柱图接近0

#### RSI 备注示例
- ✅ "满足：超卖区，可考虑做多" - RSI<30
- ✅ "满足：超买区，可考虑做空" - RSI>70
- ❌ "不满足：RSI中性区间，无明显信号" - 40-60
- 📊 "不满足：接近超卖但未达标" - 30-40

#### BOLL 备注示例
- ✅ "满足：接近下轨，支撑位做多" - 百分比<10%
- ✅ "满足：接近上轨，阻力位做空" - 百分比>90%
- ❌ "不满足：在中轨附近，无明显支撑阻力" - 40-60%

#### ATR 备注示例
- ✅ "满足：低波动，适合精细交易" - ATR<1%
- ✅ "满足：高波动，盈利空间大但风险高" - ATR>3%
- 📊 "不满足：波动偏高" - ATR 2.5-3%

## 💡 使用建议

### ✅ 什么时候入场
1. **综合评分 ≥ 60分**（至少4星）
2. **所有关键指标显示绿色** （趋势、MACD、RSI至少2个）
3. **成交量确认** (放量背景下的信号更可靠)
4. **K线形态完整** (有明确的反转信号)

### ❌ 什么时候观望
1. **综合评分 < 40分**（低于3星）
2. **趋势不明**（在均线附近震荡）
3. **MACD未确认**（在零轴附近震荡）
4. **缩量行情**（成交量不足）

### ⚠️ 风险提示
1. 🚨 **无任何操作建议**：系统仅提供信号分析，具体操作需用户自行判断
2. 🚨 **风险始终存在**：即使5星信号也有失败的概率
3. 🚨 **资金管理最重要**：严格控制单笔风险比例（建议≤2%）
4. 🚨 **止损必须设置**：根据支撑位或ATR动态设置止损
5. 🚨 **定期复盘**：记录每笔交易，持续优化入场条件

## 📞 技术支持

### 常见问题

**Q: 为什么有时没有信号？**
A: 系统严格按照条件筛选，无符合条件的信号时不会生成。这是为了提高胜率。

**Q: 红色的指标意味着什么？**
A: 红色表示该指标不满足交易条件，建议等待指标变绿后再考虑入场。

**Q: BOLL布林带位置百分比怎么理解？**
A: 0-100表示价格在上下轨之间的位置。0%=下轨，100%=上轨，50%=中轨。

**Q: 为什么同一个周期信号会变化？**
A: K线实时更新，每次更新都会重新计算所有指标，信号可能随时变化，这是正常的。

**Q: 打分80分以上的信号一定会赚钱吗？**
A: 不一定。交易有风险，即使5星信号也可能失败。关键是做好资金管理和止损。
