# 📱 大众分享版

## 简介

**大众分享版**是基于三步交易法的纯净版本，专为普通用户设计，可安全分发。

## 特点

### ✅ 安全
- **仅展示行情**：不包含任何交易 API 权限
- **无资产风险**：无法进行任何交易操作
- **合规友好**：符合各大应用市场审核要求

### 🎯 功能
- 实时 K 线图查看
- 三步交易信号检测
- AI 量化评分
- 多周期切换
- VIP 会员系统

### 📦 适用场景
- 应用市场分发
- 分享给朋友学习
- 教学演示
- 社区推广

## 与其他分支的区别

| 功能 | 3step-trading | auto-trade | public-share |
|------|--------------|-----------|--------------|
| K线图 | ✅ | ✅ | ✅ |

## APK 打包

此分支可直接打包为 APK 分发给用户：

```bash
# 安装依赖
npm install

# 同步 Android
npx cap sync android

# 打包 APK
cd android && ./gradlew assembleRelease
```
